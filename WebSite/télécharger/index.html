Comment jouer

Pour une installation de l'application sur votre ordinateur, suivez ces étapes :

  1. Téléchargez le code source de la bataille navale
  2. Installez Python 3 ou supérieur (si vous ne l'avez pas encore sur votre machine).
    Télécharger Python 3
  3. Vous pouvez maintenant lancer le fichier principal avec la commande :

    python chemin-du-dossier-source/viewer.py
